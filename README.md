### La plataforma definitiva para el seguimiento de torneos de Padel.

**Liga Pro Consulta** es la aplicación oficial de consulta para jugadores y aficionados de **Vibe Events**. Diseñada para ofrecer una experiencia fluida, rápida y moderna, permitiendo acceder a toda la información crítica del torneo en tiempo real.

---

## 🚀 Características Principales

*   **📅 Calendario de Partidos:** Visualiza horarios, pistas y enfrentamientos actualizados al minuto.
*   **📊 Resultados en Vivo:** Sigue el progreso de cada categoría y ronda.
*   **🏆 Tablas de Posiciones:** Consulta los standings reales de la temporada activa.
*   **📱 Diseño Responsive:** Optimizado para una consulta rápida desde dispositivos móviles en el club.
*   **⚡ Conectividad Premium:** Integrado con Supabase para datos instantáneos y confiables.

---

## 🛠️ Stack Tecnológico

*   **Frontend:** React + TypeScript + Vite
*   **Estilos:** CSS Moderno / Tailwind
*   **Backend:** Supabase (Database & Auth)
*   **Despliegue:** Vercel

---

## ⚙️ Configuración para Desarrolladores

Si deseas ejecutar este proyecto localmente:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/Pedro-IA-Netxtrust/liga-pro-consultas.git
    ```
2.  **Instalar dependencias:**
    ```bash
    npm install
    ```
3.  **Variables de Entorno:**
    Crea un archivo `.env` basado en `.env.example` con tus credenciales de Supabase.
4.  **Iniciar modo desarrollo:**
    ```bash
    npm run dev
    ```
